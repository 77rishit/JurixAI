✨ Overview  
Legal documents can be difficult to understand due to complex language ⚖️.  
This project uses Google Cloud and Generative AI to convert contracts into clear and simple English.

✅ Upload a document (PDF/Image/DOC) or paste your text  
✅ Click on Analyze → Get a quick, easy-to-read summary  
✅ Ask questions through chat 💬 just like talking to a lawyer  

⚡ Features  
🔍 AI-Based Legal Insights – Converts complex clauses into simple explanations  
💬 Smart Chat Assistant – Remembers context and answers your queries  
🎙️ Voice Input – Speak instead of typing  
🗣️ Voice Output – AI responds with audio  
🧑‍⚖️ Virtual AI Lawyer – Interactive experience like a real consultation  
🌍 Supports Multiple Languages – Understand content in your preferred language  
📱 Mobile-Friendly Design – Works smoothly on all devices  

🛠️ Tech Stack  

Frontend: React + Tailwind CSS  
Backend: FastAPI (Python)  
AI/ML: Google Vertex AI (Gemini), Speech Recognition, Text-to-Speech
